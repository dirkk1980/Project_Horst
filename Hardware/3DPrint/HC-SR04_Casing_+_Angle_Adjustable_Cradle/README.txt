# Summary

This is a casing & supporting cradle for HC-SR04 (Ultrasonic Distance Sensor).

<h4>Key features:</h4>
<p>
<ul type="disc">
<li>No screws or adhesive need to assemble the unit itself.</li>
<li>The vertical angle of the HC-SR04 can easily be adjusted by tilting it up or down in the cradle.</li>
<li>The horizontal angle can be adjusted or fixed depending on how many screws u use to secure it too whatever.</li>
<li>For convenience the intact casing containing the HC-SR04 can easily be removed by hand from the cradle. The cradle itself is intended to be secured  more permanently my screws.</li></ul type="disc"> </p>

<h4>Printing notes:</h4>

<ul type="disc">
